# Invite Landing Page

Host `invite-web/invite.html` on your web domain, for example:
- `https://your-domain.com/invite.html`

Update these placeholders:
1. `src/config/links.js`
- `INVITE_WEB_BASE_URL`
- `APK_DOWNLOAD_URL`

2. `invite-web/invite.html`
- `apkUrl`

Then your shared invite link can open app and fallback to APK download.
